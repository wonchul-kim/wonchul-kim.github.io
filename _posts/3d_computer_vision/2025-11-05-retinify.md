# [Retinify](https://retinify.ai/)

## Install

#### 설치하는 과정에서 `TensorRT`를 못 찾는 경우

* 명령어: 
```sh
./build.sh --install --tensorrt
``` 

* 에러 1
  ```sh
  ...
  CMake Error at retinify/CMakeLists.txt:100 (message):
    TensorRT libraries not found (x86_64)

  ...
  -- Configuring incomplete, errors occurred!

  See also "/HDD/_projects/etc/retinify/build/CMakeFiles/CMakeOutput.log".
  ```

  * 해결법:

      * 이는 TensorRT 라이브러리 경로를 CMake가 여전히 찾지 못하는 것

      * `retinify/CMakeLists.txt`를 수정해야함

      ```sh
      # TENSORRT_ROOT 환경 변수가 설정되어 있다면 힌트 경로에 추가
      if(NOT "$ENV{TENSORRT_ROOT}" STREQUAL "")
          list(APPEND _TRT_LIB_HINTS "$ENV{TENSORRT_ROOT}/lib")
          list(APPEND _TRT_INC_HINTS "$ENV{TENSORRT_ROOT}/include")
      endif()
      ```

      이 코드를 기존 find_library 및 find_path 호출 이전에 넣고, _TRT_LIB_HINTS와 _TRT_INC_HINTS 변수가 이미 설정된 표준 경로 리스트에 환경 변수 경로를 추가하도록 수정해야 함
